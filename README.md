# Homebrew Fonts Tap

Personal Homebrew tap for fonts published by [Vinit Kumar](https://github.com/vinitkumar).

## Berka Mono Closer

Install:

```sh
brew tap vinitkumar/fonts
brew install --cask font-berka-mono-closer
```

Upgrade:

```sh
brew update
brew upgrade --cask font-berka-mono-closer
```

Uninstall:

```sh
brew uninstall --cask font-berka-mono-closer
```

Upstream font repository:

```text
https://github.com/vinitkumar/berka-mono-closer
```

Berka Mono Closer is a custom build of Iosevka distributed under the SIL Open Font License 1.1. It does not include proprietary commercial font assets.
